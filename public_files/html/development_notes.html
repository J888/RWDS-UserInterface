for handling the UI, getting all the images in the bucket, and MySQL queries

Image Storage

Images are stored in a AWS S3 Bucket with name/unique-identifer = rwds-images

Metadata

Metadata on images is stored in a MySQL database in a single table.

Insert Image via Http Post Request

url = 'http://localhost:3000/insertImage'

Socket.io for server to client alerts

When a post request is made to the /insertImage endpoint, the server emits a message to the client that a weapon has been found, along with metadata pertaining to the detection
